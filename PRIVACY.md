# Privacy Policy

Payload Parameter Inspector does not collect, transmit, sell, or share personal data.

## What the extension does

The extension runs locally in the user's browser and helps parse pasted payload data, compare it against saved parameter rules, and display readable results.

## Data handling

- Payload text is processed locally inside the extension popup.
- Saved required, recommended, and optional parameter rules are stored locally using Chrome extension storage on the user's device.
- No payload data is sent to external servers by this extension.
- No analytics, tracking, or advertising SDKs are included.

## Permissions

- `storage`: used only to save the user's parameter rule lists locally.

## Contact

For questions about this extension or this privacy policy, contact the publisher through the Chrome Web Store listing or the project's GitHub repository.
