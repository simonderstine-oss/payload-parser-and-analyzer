# Quick Notes Extension

A simple Chrome extension popup for jotting down persistent notes with basic rich-text formatting.

## Features

- Bold, italic, underline
- Paragraph, H1, and H2 text styles
- Bulleted and numbered lists
- Checklist blocks with real checkboxes
- Left alignment
- Clear formatting
- Auto-save to `chrome.storage.local`

## Load in Chrome

1. Open `chrome://extensions`
2. Turn on Developer mode
3. Click `Load unpacked`
4. Select this folder: `quick-notes-extension`
